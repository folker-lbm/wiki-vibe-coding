# Folks Mirror — Plataforma de Diagnóstico Cultural

## El proyecto
Plataforma multi-método de diagnóstico cultural para consultoras. Evalúa la cultura de una organización usando múltiples métodos (no solo encuestas). Cada proyecto es diferente — la herramienta se adapta.

Primero herramienta interna de Folks, después potencialmente SaaS.

## El cliente
Folks — Culture Design Studio. 4 socios: Luka, María, Pablo, Ignacio.
Framework: Narrative → Proof → Momentum.
Paquete "The Wake-Up Call" (€25K-45K): Culture Audit + Narrative Architecture, 6-8 semanas.

## IMPORTANTE: No hay método fijo
No existe un framework de diagnóstico formalizado. La herramienta y la metodología se crean juntas. El @designer debe definir la metodología ANTES de que nadie diseñe pantallas.

## Métodos de diagnóstico

### Cuantitativos
- Encuestas/cuestionarios (escalas, índices)
- Pulse checks (micro-encuestas recurrentes)
- Análisis de datos existentes (rotación, absentismo, eNPS)

### Cualitativos
- Sensemaking (Snowden/Cynefin): micro-narrativas + self-signification
- Entrevistas semi-estructuradas
- Workshops de diagnóstico
- Observación etnográfica
- Análisis de artefactos (comunicación interna, espacios, rituales)

### Mixtos
- Triangulación (cruzar datos de diferentes métodos)
- Journey mapping participativo

## Qué resuelve la plataforma

### Para Folks (admin)
- Diseñar diagnósticos con métodos flexibles
- Recopilar datos según el método elegido
- Dashboard unificado que cruza TODOS los métodos
- Generar Culture Reality Report
- Iterar el cuestionario entre proyectos

### Para empleados (respondente)
- Experiencia diferente según método (encuesta, sensemaking, etc.)
- Anónimo, mobile-first, 10-15 min máximo
- No es un Google Forms — es una experiencia

### Para el cliente/sponsor (viewer)
- Resultados read-only, heat maps, comparativas
- Sin respuestas individuales
- Implicaciones, no solo datos

## Principios
- **Flexible, no prescriptivo**: la herramienta ofrece, Folks decide
- **Triangulación**: el valor está en cruzar métodos
- **Interpretación del participante**: inspirado en sensemaking
- **Provocador**: tono "wake-up call", no corporativo neutro

## Lo que NO es (v1)
- Self-service (Folks siempre involucrado)
- Encuesta de clima laboral genérica
- GPTW o similar
- No necesita integraciones HRIS ni IA generativa en v1
