## Summary

Overview de 10 features de Claude que la mayoría de usuarios de pago no aprovecha. Desde Projects y Deep Research hasta Claude Code y Cowork. Enfoque: "estás pagando $20/mes por un chatbot cuando podrías tener un sistema de productividad completo."

## Key Claims

- **Projects**: workspaces persistentes con memoria. Crear uno por área (Content, Work, Research, Code, Clients). La diferencia entre hablar con un extraño y trabajar con alguien de tu equipo
- **Deep Research**: botón separado (bottom left). 3-10 minutos de investigación multi-fuente con citaciones. Descompone en sub-tareas, busca en paralelo, cross-referencia claims
- **Artifacts**: ficheros interactivos renderizados en panel lateral. Dashboards, SVGs, React components, presentaciones, data viz. Más rápido que Canva/Figma para quick visuals
- **Memory**: funciona en background, editable en Settings. Importación desde ChatGPT disponible
- **Styles**: presets de tono guardados (Work, Writing, Technical, Quick). Pequeña feature, mayor ahorro de fricción diaria
- **Web Search**: toggle en input. Para todo time-sensitive, debería estar siempre on
- **Code Execution**: ejecuta Python/JS en el chat y muestra output. Análisis CSV, cálculos, parsing, conversiones
- **File Creation**: genera Word, Excel, PPT, PDF descargables directamente
- **Claude Code**: incluido en Pro ($20/mes). Terminal agent que lee codebase completo
- **[[Claude Cowork]]**: background tasks, abre apps, navega browser, Dispatch para asignar desde móvil
