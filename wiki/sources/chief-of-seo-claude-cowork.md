## Summary

Sistema de 20 prompts para SEO local usando [[Claude Cowork]] con acceso a Chrome (Ahrefs, SEMrush, Google Search Console). Demuestra el patrón de **context loading + prompt library** aplicado a un dominio específico. El autor reporta que análisis que antes tomaban medio día ahora toman minutos.

## Key Claims

- Claude Cowork accede a herramientas de SEO via Chrome extension — Ahrefs, SEMrush, GSC, Google Maps
- **Patrón clave: context loading primero** — cargar todo el contexto del negocio (nombre, servicios, competidores, keywords, GBP) antes de cualquier prompt. Sin esto, output genérico
- 20 prompts organizados en 4 partes:
  - **GBP (1-8):** categorías, atributos, reviews, respuestas a reviews, posts, servicios, descripción, fotos
  - **Website (9-13):** keyword gap, money pages, service+city pages, GSC analysis, review sentiment
  - **Backlinks (14-16):** backlink audit, citation audit, search intent mapping
  - **Content+Tracking (17-20):** content gap, entity optimization, GBP posting patterns, monthly report
- Plan de ejecución: 12 semanas, prompts ordenados de quick wins a moat competitivo
- **Entity optimization** (prompt 18) — construir presencia en el Knowledge Graph de Google via schema markup, Wikidata, brand consistency
- Review sentiment analysis para reverse-engineer el lenguaje emocional de clientes
- Una categoría secundaria añadida al GBP puede hacer aparecer en nuevas búsquedas en una semana
